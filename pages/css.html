KQM CSS Resources

Tables are a key part of a lot of KQM guides, and there are a lot of different ways that they can be used to present information. Here's a few of the different tables we've made in the past for KQM guides that you can use in transcription:

Rounded Corners Table

By Ayzel. Ping for questions or if you want any modificaations.

HTML

<table class="rounded-corners">
 <tr><th>Weapon Name</th><th>Damage</th><th>Percentage</th><th>Description</th></tr>
 <tr><td>Weapon A</td><td>50000</td><td>100%</td><td>This is a weapon</td></tr>
 <tr><td>Weapon B</td><td>50000</td><td>100%</td><td>This is a weapon</td></tr>
 <tr><td>Weapon C</td><td>50000</td><td>100%</td><td>This is a weapon</td></tr>
</table>

CSS

:root {
  --table-border-width: 2px;
  --primary-color: #29232E;
  --secondary-color: #9858C6;
  --tertiary-color: #302936;
  --quantenary-color: #504b5562;
}

.rounded-corners {
  border-collapse: separate;
  border-spacing: 0;
  min-width: 30vw;
  color: white;
}

.rounded-corners tr th,
.rounded-corners tr td {
  border-right: var(--table-border-width) solid var(--primary-color);
  border-bottom: var(--table-border-width) solid var(--primary-color);
  padding: 5px;
}

.rounded-corners tr th:first-child,
.rounded-corners tr td:first-child {
  border-left: var(--table-border-width) solid var(--primary-color);
}

.rounded-corners tr th {
  background: var(--tertiary-color);
  border-top: var(--table-border-width) solid var(--primary-color);
  text-align: left;
}

.rounded-corners tr td {
  background-color: var(--quantenary-color);
}

.rounded-corners tr:first-child th:first-child {
  border-top-left-radius: 5px;
}

.rounded-corners tr:first-child th:last-child {
  border-top-right-radius: 5px;
}

.rounded-corners tr:last-child td:first-child {
  border-bottom-left-radius: 5px;
}

.rounded-corners tr:last-child td:last-child {
  border-bottom-right-radius: 5px;
}

KQM uses images in basically all guides, and we make sure to properly credit artists. Here's some quick CSS that allows you to put artist credits in the bottom corner of an image.

Lati Images

By Lati. Ping if you have questions.

Instructions: Just apply the CSS class to an image. Put the Artist credits into the caption (in Wordpress). It should work automatically. Use either [lati-image] or [lati-image-left] (for the left side), not both at the same time.

CSS

/* Artist Images */
.lati-image img, .lati-image-left img {
   border-radius: 25px;
   display: block;
}

.lati-image figcaption, .lati-image-left figcaption {
   padding: 5px 20px 5px 20px;
   background-color: #000000;
   top: -36px;
   display: inline-block;
   position: relative;
}

.lati-image figcaption {
   border-radius: 20px 0px 18px 0px;
   float: right;
}

.lati-image-left figcaption {
   border-radius: 0px 20px 0px 18px;
   float: left;
}

Carousels are a cool way to display things like teams. Carousels can be modified in many ways, although creating carousels in pure HTML/CSS is pretty difficult. I've provided an example carousel below for use. Note that if you want to use the carousels in this section, you will have to either a] contact me (Ayzel) or b] be familiar with how to use custom HTML blocks, as this carousel is entirely custom HTML - no Wordpress blocks used.

Vertical Carousel

By Ayzel. Ping if you have any questions.

/* carousel css */
#carousel-container {
   display: flex;
   justify-content: center;
   align-items: center;
   min-height: 500px;
}

#carousel, #carousel-navigation {
   flex: initial;
   display: block;
}

#carousel h1 {
   font-size: 32px;
   line-height: 42px;
}

#carousel h2 {
   font-size: 26px;
   line-height: 36px;
}

#carousel p {
   font-size: 16px;
   line-height: 26px;
}

#carousel {
   overflow: hidden;
   width: 600px;
   height: 400px;
   border-radius: 4px;
   background-color: #fff;
}

.p-wrapper {
   position: relative;
   width: 100%;
   height: 100%;
   transition: transform 0.4s ease-in-out;
}

#button_a:checked ~ #carousel .p-wrapper {
   -webkit-transform: translateY(0);
   transform: translateY(0);
}

#button_b:checked ~ #carousel .p-wrapper {
   -webkit-transform: translateY(-100%);
   transform: translateY(-100%);
}

#button_c:checked ~ #carousel .p-wrapper {
   -webkit-transform: translateY(-200%);
   transform: translateY(-200%);
}

#button_d:checked ~ #carousel .p-wrapper {
   -webkit-transform: translateY(-300%);
   transform: translateY(-300%);
}

   height: inherit;

   margin-left: 24px;
}

}
