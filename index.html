About This Site

Ayzel here. KQM's editor resources are very scattered, which is something that numerous editors have noted. This makes it difficult to find even basic things, and makes learning CSS impossible, which also screws with any kind of guide CSS standardization we want to do. This website is designed to be a repository of all KQM website resources; people can look here if they're just starting their editor journey or if they're experienced editors wanting access to other editor-designed CSS elements/want to just refresh their memories about the basics.
